## Android Vibration
-unity에서 android vibration관련 설정에 필요한 파일이다  

## Usage
-해당 프로젝트 폴더의 Asset\Plugins\Android에 넣는다.    
-Vibration.cs를 사용한다.  

## Notice
-Unity 2021버전은 IOS와 Android의 Vibration이 통합되었다.  
-또한 파일명이 UnityManifest.xml로 바뀌었으니 경로에 들어가서 찾아보자
